Simple Window batch automation script for problems that require you to run chkdsk command and reboot multiple times.



Additionally, you can put this in the " shell:common startup " folder to run this script on every reboot automatically.

Version 1.0.1:

Works the same way as before but now automatically elevates to administrator privileges when run (now you do not have to click on 'run as administrator' anymore).
